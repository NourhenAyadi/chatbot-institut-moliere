# Vocabulaire Français par Thèmes pour les Enfants

Apprendre le vocabulaire par thèmes permet de mémoriser les mots dans leur contexte. Voici les thèmes essentiels pour les débutants.

## Thème 1 : La Famille (La famille)
- le père / le papa (father / dad)
- la mère / la maman (mother / mom)
- le frère (brother)
- la sœur (sister)
- le grand-père / le papi (grandfather / grandpa)
- la grand-mère / la mamie (grandmother / grandma)
- l'oncle (uncle)
- la tante (aunt)
- le cousin / la cousine (cousin)
- **Phrase exemple** : "Ma famille est grande. J'ai un frère et deux sœurs."

## Thème 2 : Les Couleurs (Les couleurs)
- rouge (red), bleu (blue), vert (green), jaune (yellow)
- orange (orange), rose (pink), violet (purple)
- noir (black), blanc (white), gris (grey), marron (brown)
- **Phrase exemple** : "Le ciel est bleu. Les feuilles sont vertes."

## Thème 3 : Les Chiffres (Les chiffres)
- 1 = un, 2 = deux, 3 = trois, 4 = quatre, 5 = cinq
- 6 = six, 7 = sept, 8 = huit, 9 = neuf, 10 = dix
- 20 = vingt, 30 = trente, 50 = cinquante, 100 = cent
- **Phrase exemple** : "J'ai cinq crayons. Il y a dix élèves dans la classe."

## Thème 4 : L'École (L'école)
- la classe (classroom), le tableau (blackboard), le bureau (desk)
- le cahier (notebook), le stylo (pen), le crayon (pencil)
- le livre / le manuel (book/textbook), la trousse (pencil case)
- le professeur / le maître / la maîtresse (teacher)
- l'élève (student/pupil)
- **Phrase exemple** : "Je mets mon livre et mon cahier dans mon sac."

## Thème 5 : La Nourriture (La nourriture)
- le pain (bread), le lait (milk), le fromage (cheese)
- les fruits : la pomme (apple), la banane (banana), l'orange (orange)
- les légumes : la carotte (carrot), la tomate (tomato), la salade (lettuce)
- le poulet (chicken), le poisson (fish), les œufs (eggs)
- **Phrase exemple** : "Ce matin, je mange du pain avec du fromage. Je bois du lait."

## Thème 6 : Le Corps Humain (Le corps)
- la tête (head), les yeux (eyes), le nez (nose)
- la bouche (mouth), les oreilles (ears), les bras (arms)
- les mains (hands), les jambes (legs), les pieds (feet)
- **Phrase exemple** : "J'ai mal à la tête. Mes mains sont froides."

## Thème 7 : Les Animaux (Les animaux)
- le chien (dog), le chat (cat), le lapin (rabbit)
- l'oiseau (bird), le poisson (fish), le cheval (horse)
- le lion (lion), l'éléphant (elephant), le singe (monkey)
- **Phrase exemple** : "Mon animal préféré est le chat. Il est doux et calme."

## Thème 8 : Les Salutations et Expressions Courantes
- Bonjour ! / Bonsoir ! (Hello! / Good evening!)
- Salut ! (Hi! - informal)
- Merci / Merci beaucoup (Thank you / Thank you very much)
- S'il vous plaît / S'il te plaît (Please - formal/informal)
- Excusez-moi / Pardon (Excuse me)
- Oui / Non / Peut-être (Yes / No / Maybe)
- Je ne comprends pas (I don't understand)
- Pouvez-vous répéter ? (Can you repeat, please?)
